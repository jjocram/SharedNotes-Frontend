# SharedNotesFrontend

This is the frontend for the [SharedNotes Backend](https://github.com/jjocram/SharedNotes-Backend).

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.2.9.

## Run it

To run this project you can:

- configure your IDE (tested on WebStorm) and click "Run"
- excute the following commands on the terminal:
  1. `npm install` (only the first time)
  2. `ng serve`

Then you can open a web browser and go to [http://localhost:4200](http://localhost:4200)
